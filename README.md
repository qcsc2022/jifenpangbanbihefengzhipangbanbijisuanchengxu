## 程序功能

该程序实现了以下功能：

1. **积分旁瓣比（ISLR）计算**：计算信号的积分旁瓣比，用于衡量信号主瓣与旁瓣之间的能量分布。
2. **峰值旁瓣比（PSLR）计算**：计算信号的峰值旁瓣比，用于评估信号旁瓣的最大峰值与主瓣峰值之间的比值。
